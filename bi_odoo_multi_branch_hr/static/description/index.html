Human Resource Multiple Branch(Unit) Operation Setup Odoo Apps for Single Company.

Do you want to manage multiple branch units of HR management for single company? here you go, this Odoo apps helps users to make different branch for single company and use that branch for Human Resource management with multi-branch concept which works same as multi-company environment.This apps is depends on multiple branch apps which works for all generic application i.e Contact, Products, Sales,Purchase, Billing, Warehouse, Accounting , Point of Sale etc.

Features

Set Branch on Employee.

You can set Branch on Employee, HR Department, Attendance, Payslip, Expense, Expense Sheet, Contract, Holidays.

View Branch on PDF and Analysis Report.

You can view Branch on PDF report of payslip, and Analysis Report of Expence and Expense Expenses Reports.
Demo Request Video Tutorial

Create Multiple Branch

You can create Multiple Branch goto Settings > Users > Branch.

Assigning Branch

You can assign a branch for which specific user is belongs.

Branch On Departments

Manager can set branch in department form.

Branch On Job Applicant Form

Manager can set branch in job applicant form based on applied job.Click on create employee button to create employee.

Branch on Employee

While create employee, Branch is also set as employee branch.

Branch on Employee Attendance

While creating employee attendance, Branch is also set from employee branch.

Branch on Employee Contract

While creating employee contract, Branch is also set from employee branch.

Branch on Employee Payslip

While creating employee payslip, Branch is also set from employee branch.

Branch on Expense

Manager can set branch on employee expense form.

Branch on Expense Sheet

While creating expense sheet, Branch is also set from employee expense.

Branch on Expense Analysis Report

You can aslo view branch on Expenses Analysis.

Expenses Reports Analysis Report.

Payslip Details Report

You can aslo view branch on report of payslip details.
